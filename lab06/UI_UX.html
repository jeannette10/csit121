Website Evaluation

Website Information

URL: https://umd.edu/

Name: University of Maryland

Target Audience

The target audience includes prospective students, current students, faculty, staff, alumni, and visitors interested in learning more about the university's programs, research, and campus life.

Site Organization

The site is organized into several main sections, including About, Academics, Admissions & Aid, Student Life, Research, and News. Each section contains subpages that provide detailed information relevant to the topic.

CRAP Design Principle

The site uses the Contrast principle effectively. For example, the homepage features bold, contrasting colors to highlight important information and calls to action, making it easy for users to navigate and find key details.

Accessibility Audit Score

To find the audit score, you can use tools like WAVE or Lighthouse. These tools will provide a detailed report on the site's accessibility, including any issues that need to be addressed.

Site Effectiveness

The site supports users in completing actions accurately by providing clear navigation, well-organized content, and accessible information. Users can easily find information about admissions, programs, and campus events.

Site Efficiency

Users can perform tasks quickly due to the intuitive layout and search functionality. The site is designed to minimize the number of clicks needed to reach important information.

Engagement

The site is pleasant to use and appropriate for its industry/topic. It features engaging visuals, interactive elements, and up-to-date content that keeps users informed and interested.

Recommendation

One recommendation to improve the website would be to enhance the mobile responsiveness. While the site is generally accessible, optimizing it further for mobile devices would ensure a seamless experience for users accessing it on smartphones and tablets.
